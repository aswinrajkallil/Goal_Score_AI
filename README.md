## 🌟 Overview

GoalScore AI is a full-stack web application built to help football enthusiasts explore the FIFA World Cup 2026 through an intelligent conversational interface.

Users can ask football-related questions, explore standings, understand qualification scenarios, and receive AI-generated insights about teams and tournament developments.

---

## ✨ Features

### 🤖 AI Football Assistant
- Ask natural language questions about the FIFA World Cup 2026.
- Receive AI-powered responses tailored to football topics.
- Follow-up suggestions for deeper exploration.

### 🏆 Tournament Standings
- View World Cup group standings.
- Track team positions and points.
- Explore qualification possibilities.

### 📅 Match Information
- Access World Cup fixtures and schedules.
- Stay informed about upcoming matches.

### 📊 Intelligent Analysis
- Qualification scenario explanations.
- Team performance discussions.
- Tournament insight generation.

### 📱 Responsive User Interface
- Optimized for:
  - Desktop
  - Tablets
  - Mobile devices

### ⚡ Modern Experience
- Fast and lightweight application.
- Smooth navigation and interactions.
- World Cup-inspired interface.

## ⚙️ Getting Started

### Prerequisites

Ensure you have installed:

- Node.js (v18 or later recommended)
- npm

---

## 🔧 Installation

Clone the repository:

```bash
git clone https://github.com/aswinrajkallil/Goal_Score_AI.git
```

Navigate into the project directory:

```bash
cd Goal_Score_AI
```

---

## Frontend Setup

Navigate to the frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
VITE_API_URL=your_backend_url
```

Start the development server:

```bash
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```

---

## Backend Setup

Navigate to the backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
OPENROUTER_API_KEY=your_api_key
PORT=5000
```

Start the server:

```bash
npm run dev
```

Backend runs on:

```text
http://localhost:5000
```
